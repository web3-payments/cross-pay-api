### Compiling

* Linux:

    ```
    ./mvnw clean install dockerfile:build
    ```
* Windows:

    ```
    ./mvnw.cmd clean install dockerfile:build
    ```
  
_this command generate the image_

### Running

* Running the containers

```
docker-compose up
```

* destroying the containers

```
docker-compose down
```
